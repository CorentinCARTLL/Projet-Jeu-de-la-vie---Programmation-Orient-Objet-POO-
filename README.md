# Projet-Jeu-de-la-vie---Programmation-Orient-Objet-POO
[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

Simulation du Jeu de la Vie de Conway en programmation orientée objet (C++). Chaque cellule est un objet interagissant dans une grille évolutive. Visualisation graphique et logique respectant les règles de naissance, survie et mort des cellules.
